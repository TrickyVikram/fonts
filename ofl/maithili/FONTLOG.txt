Basic Font Information

Maithili is a Unicode typeface family that supports languages that use
the Devanagari script, specifically optimized for the Maithili language.

The Maithili project aims to provide a comprehensive set of fonts for 
the Maithili language community. The font includes:

- Full Devanagari character support for Maithili
- Traditional letterforms respecting cultural aesthetics
- Proper rendering of complex conjuncts
- Optimized metrics for readability

More specifically, this release supports the following Unicode ranges:

- Devanagari (U+0900-U+097F)
- Devanagari Extended (U+A8E0-U+A8FF)
- Vedic Extensions (U+1CD0-U+1CFF)
